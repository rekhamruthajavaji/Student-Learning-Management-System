\# 🎓 Student Learning Management System



\## Project Description

An SQL-based Student Learning Management System developed using MySQL. The project demonstrates database design, normalization, relationships, analytics, reporting, stored procedures, and views.



\## Technologies Used

\- MySQL

\- SQL



\## Features

\- Students, Courses, Instructors, Enrollments, Assignments, and Results Management

\- Primary and Foreign Key Relationships

\- Normalization up to 3NF

\- Top 5 Students by Average Marks

\- Top 5 Popular Courses by Enrollment

\- Semester Performance Reports

\- Stored Procedures for Student Enrollment and Result Generation

\- Academic Dashboard View
